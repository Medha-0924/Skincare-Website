# 🌿 Skincare E-Commerce Website

A full-stack skincare e-commerce web application built using modern web technologies, designed to provide users with a smooth, secure, and elegant online shopping experience.

## 💡 Features

- 🔐 **User Authentication & Authorization**
  - Secure login/signup with password hashing using **bcrypt**
  - Token-based authentication using **JWT**
  - Cookie management with **cookie-parser**

- 🛒 **E-Commerce Functionality**
  - Add to Cart
  - Integrated **Payment Gateway** for seamless checkout

- 🧩 **Full-Stack Integration**
  - **Frontend:** React, HTML, CSS, JavaScript
  - **Backend:** Node.js, Express.js
  - **Database:** MongoDB

- 🛡️ **Security & Config**
  - Environment management with **dotenv**
  - Secure data exchange with **CORS** enabled

## 📁 Tech Stack

| Frontend        | Backend             | Database |
|-----------------|---------------------|----------|
| React.js        | Node.js + Express.js| MongoDB  |
| HTML + CSS      | JWT, bcrypt         |          |
| JavaScript      | dotenv, CORS        |          |
